## 🎮 Sobre el Juego: Mirage Realms
**Mirage Realms** es un MMORPG indie retro en 2D desarrollado para Android y PC. Destaca por su estética "Zelda-like" de 16 bits y su mundo abierto persistente.
- **Sin Límite de Nivel:** Progresión infinita basada en el uso de habilidades.
- **Combate de la Vieja Escuela:** Un sistema de combate táctico donde cada movimiento cuenta.
- **Clases Únicas:** Elige entre Caballero, Guardabosques, Mago o Chamán.
- **Economía Real:** Comercio entre jugadores y recolección de recursos.

---

## 🚀 Características Principales

### ⚔️ Auto-Attack (Modo Caza)
Optimiza tu tiempo de juego con un sistema de ataque automatizado diseñado para la máxima eficiencia.
- **Detección Inteligente:** Identifica mobs en pantalla mediante visión artificial.
- **Intervalos Configurables:** Ajusta la velocidad de ataque para imitar el comportamiento humano.
- **Optimización de Target:** Prioriza enemigos según proximidad y estado.

### 💊 Auto-Heal & Auto-Mana (Soporte Vital)
Nunca vuelvas a morir en una zona de alto nivel. El bot gestiona tu supervivencia de forma autónoma.
- **Jerarquía de Pociones:** Uso inteligente de consumibles desde *Weak Potion* hasta *Soul Amphora*.
- **Umbrales Personalizables:** Configura el porcentaje exacto de HP/Mana para activar la recuperación.
- **Gestión de Inventario:** Selecciona automáticamente el mejor recurso disponible en tu inventario.

### 🗺️ Grabación de Rutas
- **Patrullaje Inteligente:** Graba tus rutas de farmeo favoritas y deja que el bot las repita sin fin.
- **Alertas de Peligro:** Sistema de notificaciones ante la presencia de otros jugadores o PKs (Player Killers).

## 🛠️ Tecnología y Arquitectura

El bot está construido sobre una base tecnológica robusta para garantizar estabilidad y rendimiento:

- **Kotlin:** El lenguaje principal, aprovechando su seguridad y expresividad.
- **OpenCV:** Utilizado para el análisis de imagen en tiempo real, permitiendo al bot "ver" el mundo del juego.
- **Accessibility Service:** Interactúa con la interfaz de Android sin necesidad de acceso root, simulando toques y gestos naturales.
- **Overlay Service:** Una interfaz flotante minimalista que te permite controlar el bot directamente desde el juego.

## 📦 Instalación

1. Clona el repositorio:
   ```bash
   git clone https://github.com/samatha/mirageBot.git
   ```
2. Importa el proyecto en **Android Studio**.
3. Compila e instala el APK en tu dispositivo Android.
4. Habilita el **Servicio de Accesibilidad** para `mirageBot` en los ajustes de tu teléfono.
5. Abre Mirage Realms y activa los módulos deseados desde el panel flotante.
